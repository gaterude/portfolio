# Birthday Invitation Page

## Description
A HTML page to invite friends to a birthday party. The page includes:
- Party details (day, location)
- Requirements (dress code, arrival time, gifts)
- RSVP form for guests to confirm attendance and suggest activities
- A friendly birthday image to make the invitation more appealing

## How to Use
1. Open `submissn.html` in any modern web browser.
2. View the party details and requirements.
3. Fill out the RSVP form with your:
   - Name
   - Attendance confirmation
   - Whether you'll bring company
   - Activity suggestions (optional)
4. Submit the form.

## Features
- Responsive design for mobile and desktop
- Simple, clean layout
- Interactive RSVP form with required fields
- Image included for visual appeal

## Requirements
- Modern web browser 
- Internet connection
